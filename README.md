概述
====

Equinox 是 Sandwych Consulting 为 OpenERP 软件开发的一整套汉化与本地化模块包。
Equinox 也是我们的 WychERP 产品的一个基本组成部分。

Equinox 包含如下模块：

* sandwych_cn_pdf: PDF 报表中文字体支持
* l10n_cn_hr_payroll: 中国薪酬模块
* l10n_cn_account_report: 中国财务报表
